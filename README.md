# To-Do List Application

Một ứng dụng quản lý công việc hiện đại, tối ưu hóa quy trình làm việc cá nhân và cộng tác nhóm.

## Features
- **Kanban Board:** Kéo thả trực quan để thay đổi trạng thái công việc.
- **Calendar Integration:** Xem tổng quan kế hoạch và quản lý deadline dễ dàng trên lịch. 
- **Real-time updates using SignalR:** Đồng bộ hóa dữ liệu tức thì không cần tải lại trang.
- **Task management & collaboration:** Phân quyền, giao việc, gắn thẻ, và cập nhật trạng thái làm việc nhóm.

## Technologies
- ASP.NET Core MVC
- SignalR
- SQL Server
